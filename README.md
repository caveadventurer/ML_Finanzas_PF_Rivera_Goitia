- Este repositorio contiene el desarrollo completo del proyecto final de la asignatura Introducción a la Ciencia de Datos y Machine Learning, aplicado al análisis y predicción del nivel de riesgo crediticio utilizando datos financieros.
- El objetivo del proyecto es comparar distintos modelos de clasificación y evaluar su capacidad para predecir si un préstamo será “exitoso” o “no exitoso”, considerando métricas de desempeño, interpretabilidad y aplicabilidad en contextos reales del sector financiero.
